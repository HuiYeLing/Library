# 用户管理接口文档

## 基础路径
`/api/user`

### 1. 通过ID获取用户信息
- **接口地址**: `/getUserById`
- **请求方式**: `GET`
- **请求参数**:
  | 参数名 | 类型   | 是否必填 | 描述       |
  |--------|--------|----------|------------|
  | id     | long   | 是       | 用户ID     |
- **响应示例**:
  ```json
  {
    "code": 200,
    "message": "成功",
    "data": {
      "id": 1,
      "username": "test_user",
      "email": "test@example.com"
    }
  }
  ```

---
