# 用户管理接口文档

### 1. 用户登录

- **接口地址**：`POST /api/user/login`
- **请求参数**（`application/x-www-form-urlencoded` 或 query params）：
  - `username` (string) 用户名（必填）
  - `password` (string) 密码（必填）
- **返回示例**：
  ```json
  {
    "code": 200,
    "msg": "登录成功",
    "data": {
      "id": 1,
      "username": "admin",
      "role": "admin",
      "token": "xxxxxx"
    }
  }
  ```

---

### 2. 用户注册

- **接口地址**：`POST /api/user/register`
- **请求参数**（`application/x-www-form-urlencoded` 或 query params）：
  - `username` (string) 用户名（必填）
  - `password` (string) 密码（必填）
  - `email` (string) 邮箱（必填）
- **返回示例**：
  ```json
  {
    "code": 200,
    "msg": "注册成功",
    "data": null
  }
  ```

---

### 3. 通过ID获取用户信息

- **接口地址**：`GET /api/user/getUserById`
- **请求参数**：
  - `id` (long) 用户ID（必填）
- **返回示例**：
  ```json
  {
    "code": 200,
    "msg": "查询成功",
    "data": {
      "id": 1,
      "username": "admin",
      "email": "admin@xxx.com",
      "role": "admin"
    }
  }
  ```

---

### 4. 通过用户名获取用户信息

- **接口地址**：`GET /api/user/getUserByUsername`
- **请求参数**：
  - `username` (string) 用户名（必填）
- **返回示例**：
  ```json
  {
    "code": 200,
    "msg": "查询成功",
    "data": {
      "id": 1,
      "username": "admin",
      "email": "admin@xxx.com",
      "role": "admin"
    }
  }
  ```

---

### 5. 删除用户

- **接口地址**：`DELETE /api/user/deleteUser`
- **请求参数**：
  - `id` (long) 用户ID（必填）
- **返回示例**：
  ```json
  {
    "code": 200,
    "msg": "删除成功",
    "data": null
  }
  ```

---

### 6. 获取所有用户

- **接口地址**：`GET /api/user/getAllUsers`
- **请求参数**：无
- **返回示例**：
  ```json
  {
    "code": 200,
    "msg": "查询成功",
    "data": [
      {
        "id": 1,
        "username": "admin",
        "email": "admin@xxx.com",
        "role": "admin"
      },
      {
        "id": 2,
        "username": "user1",
        "email": "user1@xxx.com",
        "role": "user"
      }
    ]
  }
  ```

---

### 7. 插入用户

- **接口地址**：`POST /api/user/insertUser`
- **请求参数**（`application/x-www-form-urlencoded` 或 query params）：
  - `username` (string) 用户名（必填）
  - `password` (string) 密码（必填）
  - `email` (string) 邮箱（必填）
  - `created_at` (string) 创建时间（必填，格式如 `2024-06-04 12:00:00`）
- **返回示例**：
  ```json
  {
    "code": 200,
    "msg": "插入成功",
    "data": null
  }
  ```

---

### 8. 获取当前登录用户信息

- **接口地址**：`GET /api/user/getCurrentUser`
- **请求参数**：无
- **返回示例**：
  ```json
  {
    "code": 200,
    "msg": "查询成功",
    "data": {
      "id": 1,
      "username": "admin",
      "email": "admin@xxx.com",
      "role": "admin"
    }
  }
  ```

---

**说明：**

- 所有接口返回统一的 `ApiResult` 格式，包含 `code`、`msg`、`data` 字段。
- 登录、注册、插入用户等接口均为 `POST`，参数通过 `@RequestParam` 传递（即表单或 query）。
- 删除用户为 `DELETE`，其余查询为 `GET`。
- 登录接口返回的 `data` 中包含 `role` 字段，可用于前端权限判断。
